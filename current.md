[!DNL Adobe System Status] 提供有關 Adobe 產品和服務的詳細資訊、狀態更新和電子郵件通知。收到有關服務中斷、作業中斷和維護事件的通知。請前往 [status.adobe.com](https://status.adobe.com/) 了解詳情。

+++發行說明

發行日期： **2025年7月16日**

**新增功能**

* 虛擬助理一般可用性
* 事件ID搜尋
* Slack通知設定更新
* 錯誤修正與改善功能

| 功能 | 說明 |
| ------- | ------- |
| **虛擬助理一般可用性** | <ul><li>**[!UICONTROL 虛擬助理]**&#x200B;可讓網頁或行動裝置上的Adobe Status客戶在不離開首頁的情況下尋找答案並執行快速動作。 這是一個可快速取得說明及完成工作的位置。 </li><li>虛擬助理現在可供所有Adobe Status客戶使用。 可透過登入[status.adobe.com](https://status.adobe.com/)並與&#x200B;**[!UICONTROL 詢問我們的助理]** Widget互動來存取它。</li></ul> |
| **事件識別碼搜尋** | <ul><li>客戶現在可以在產品和雲端頁面上以及虛擬助理上依事件ID進行搜尋。</li></ul> |
| **Slack通知安裝程式更新** | <ul><li>在客戶可以繼續設定之前，他們會被要求輸入其Slack管道的詳細資料。 此更新可確保順利地設定Slack，好讓Slack管道派上用場。</li></ul> |
| **錯誤修正和改善功能** | <ul><li>遺漏通知 — 針對客戶的報表可用性通知未以過去的CSO傳送給客戶</li><li>遺失通知 — 當兩個事件同時發生時，會遺失電子郵件通訊。</li><li>維護作業40天歷史記錄 — 由於某些維護作業超過40天，因此維護作業遺失歷史事件。</li><li>雲端頁面橫幅 — 在雲端頁面上變更篩選器時，橫幅沒有更新。</li><li>UI增強功能 — 與針對客戶的宣告相關的訊息已更新，以提高清晰度。</li></ul> |

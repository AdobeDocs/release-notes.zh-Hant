## Experience Cloud

Experience Cloud 中的新功能及修正。

### Marketing Cloud ID 服務 {#mcvid}

此次發行將 [!DNL  Marketing Cloud] ID 服務代碼 (` VisitorAPI.js`) 推進至 2.0 版。此版本含有下列功能、修正及其他變更。

>[!IMPORTANT]
>
>ID 服務代碼 v2.0 依預設會自動與 [!DNL  Adobe Advertising Cloud] 同步 ID。另請參閱[了解 ID 同步和比對率](https://docs.adobe.com/content/help/en/id-service/using/home.html/mcvid-match-rates.html)。

修正:

* 已修正致使 AppMeasurement 無法對 Analytics 進行追蹤呼叫的錯誤。(MCID-254、MCID-256、MCID-286)
* 修正若訪客啟用廣告封鎖程式而該封鎖程式被設定為排除 [!DNL  demdex.net] 網域時，ID 服務無法立即失效的錯誤。這是相當罕見且不尋常的錯誤，因為大多數的廣告封鎖工具不會封鎖 [!DNL  demdex.net] 網域。(MCID-233)
* 已修正 ID 服務程式碼與客戶網站上的自訂指令碼之間的互動所造成的錯誤。此問題導致 Internet Explorer 9 無法載入網頁。(MCID-206)

Marketing Cloud ID服務說明位於 [這裡](https://docs.adobe.com/content/help/zh-Hant/id-service/using/home.html)。

### [!UICONTROL Analysis Workspace] {#aa-wksp}

[!UICONTROL Analysis Workspace] 的功能和修正。

#### [!UICONTROL 同類群組分析]

[!UICONTROL 同類群組分析]的重大改善可讓您：

* 分別套用區段包含和回訪量度。
* 顯示客戶流失而非保留率。
* 顯示延遲表（包含事件前後經過的時間）。
* 自訂同類群組維度（以根據eVar而非只限時間對訪客分組）。
* 滾動式同類群組計算：根據先前時段 (而非原始同類群組) 計算保留率和流失率。
* 在包含和傳回欄位中新增多個量度，以及套用區段。 （不支援計算量度。）

#### [!UICONTROL 檢視密度]

此新設定透過減少左側欄、自由表格和同類群組表格中的垂直邊框間距，讓您可在單一畫面上查看更多資料。

This setting is accessible via **[!UICONTROL Project]** > **[!UICONTROL Project Info &amp; Settings]**.

#### 支援歸因 IQ 中的多值變數

Analytics 的部分維度可包含單一點擊的多個數值，例如 listVar、產品變數、清單 prop 或銷售 eVar。Analysis Workspace 可讓您將歸因 IQ 套用至這些點擊層級變數。

#### [!UICONTROL 行銷管道]增強功能

現在，我們將 AMO ID 和 AMO EF ID 公開至行銷管道處理規則。這些是 [!DNL Advertising Cloud] 和 [!UICONTROL Advertising Analytics] 整合使用的主要和次要追蹤程式碼。

#### 效能改進

加速細分視覺化的改進。 例如，包含多個劃分的專案載入速度更快。

#### [!UICONTROL Analysis Workspace] 中的修正

* 修正將元件拖放至面板拖放區所建立的內部區段定義不會跨作業持續存在的問題。 (AN-172110)
* 修正無法儲存對視覺化標籤進行編輯的問題。 (AN-171119、AN-170287、AN-169721)
* 修正分析工作區 [!UICONTROL 的效能問題]，尤其是包含多行視覺化的專案。 (AN-169916、AN-169691、AN-167120、AN-168082)
* 修正 **[!UICONTROL Reports &amp; Analytics]** 中[!UICONTROL 「在 Workspace 中試用」]選項的問題：拖放區段，但區段卻未顯示於對應的 Workspace 專案。(AN-169491)
* 修正當專案上有多個日期範圍時，計算量度傳回錯誤的問題。 (AN-169057)
* 修正當最上方表格收合且日期範圍變更時，劃分資料無法更新的問題。 (AN-168695)

## [!DNL Advertising Cloud] {#adcloud}

[!DNL Adobe Advertising Cloud] 中的新功能。

### 產品組合

（測試版）新的「每週」支出策略可讓您在嘗試花費指定每週金額的同時，將加權收入最大化。 這個策略會自動使用星期幾模式與模擬。

如果您想要將此策略用於投資組合，請等到該投資組合最佳化至少兩週，然後主動監控支出和效能，直到該功能被驗證為止。

您現在可以使用新的Advertising Cloud支出策略「Google Target CPA」和「Google Target ROAS」，讓Google Ads將多個促銷活動最佳化為單一廣告支出報酬率(ROAS)或每次贏取成本(CPA)目標。 透過這些支出策略，所有指派的Google Ads促銷活動都將具有指定的競標策略和指定的目標；會忽略其他搜尋引擎上的促銷活動。 所有其他作品集設定都會停用。

在「自動最佳化競標調整值」的投資組合區段中，「對象目標」選項現在會在促銷活動和廣告群組層級最佳化Google Ads再行銷清單、客戶符合和類似對象（您可在Google Ads中建立）。 在舊版中，「再行銷清單」選項僅在廣告群組層級最佳化再行銷清單。

對於啟用學習功能的作品集，Advertising Cloud現在可以在指定的學習預算內，自動以零印象的出價單位出價。

依預設，您可以使用舊式手動規則，將曝光率為零的競標單位競標，直到淘汰手動選項為止。 在這種情況下，學習預算僅適用於具有模型的競標單位，而零印象學習則使用額外支出。 作品集可能會花費超過所分配的學習預算，以符合指定的參數。

### 搜尋促銷活動

促銷活動（舊版）檢視不再位於主功能表中。 若要使用舊版檢視（將於2月停用），請前往「搜尋>促銷活動>促銷活動」，捲動至頁面右下方，然後按一下「舊版促銷活動」。

如果您位於舊版檢視中，則可從主功能表開啟「搜尋>促銷活動>促銷活動」，或捲動至右側並按一下此處，以嘗試新的促銷活動體驗，以返回新的檢視。

（僅限Yandex帳戶）當帳戶由代理／管理帳戶管理時，您現在可以使用「MCC帳戶」欄位指定管理帳戶。 要刪除現有關聯，請選擇「無MCC帳戶」。

(Google Ads)您現在可以建立回應式搜尋廣告，這些廣告在Google Ads中處於測試模式，用於搜尋和顯示網路上的促銷活動。 Google Ads會從一組廣告標題和說明動態組合文字型的互動式搜尋廣告，並偏愛搭配運用的組合。 您可以選擇將廣告標題和說明釘選至特定位置。 報表位於廣告層級。

管理回應式搜尋廣告的大量表單功能將於未來版本中提供。產品組合變更歷史記錄現在會納入指派給產品組合的促銷活動相關資料，可在「產品組合」視圖中找到該資料。要比較兩個日期範圍的資料時，您現在可以依據比較欄 (例如：曝光次數 R2) 排序資料。

錯誤訊息現在可以選擇報告問題。 工作階段資料會自動包含在報表中，您也可以選擇輸入文字詳細資訊。

## [!DNL Adobe Cloud Platform]{#acp}

Experience Cloud 介面與[!UICONTROL 平台]核心服務發行說明。包含 Mobile Services、Launch，由 Adobe 提供、動態標籤管理、GDPR API，以及 Experience Cloud ID 服務。

### [!UICONTROL 動態標籤管理]

[!DNL Adobe] 預計於 2020 年底淘汰[!UICONTROL 動態標籤管理]。

* **2019-07-01：** 從明年七月開始，DTM 將不再允許建立新屬性。這對現有屬性沒有影響。
* **2020-07-01：** DTM 屬性將會進入唯讀模式。您將無法建立或編輯工具、規則或資料元素。您將無法再發佈至 DTM 環境。這對先前發佈的程式庫沒有影響。
* **2020-12-31:** 終止支援。伺服器將解除運作、文件將離線，社群將被移除。這對先前發佈的程式庫沒有影響。

如需詳細資訊，請參閱 [DTM Plans for a Sunset](https://forums.adobe.com/community/experience-cloud/platform/launch/blog/2018/10/05/dtm-plans-for-a-sunset)。

**修正、改良及淘汰內容**

「傳出歷史報表」的名稱已更新為「傳出檔案歷史報表」。先前的名稱使某些客戶認為，此報表會顯示 HTTP 目的地的傳出資料。實際上，此報表僅顯示傳送至 S3 或 FTP 位置的檔案。

**已知問題**

* 最新版本的 Safari 包含智慧型追蹤防護 (ITP) 2.0 工具。這會影響 Safari 使用者的「可定址對象」量度，以及使用 h_referer 訊號的資料集合。請參閱相關文件，瞭解[「可定址對象」低符合率成因](https://marketing.adobe.com/resources/help/en_US/aam/addressable-audiences-match-rates.html)的 Safari 流量，以及[使用 h_ 前綴的資料集合](https://marketing.adobe.com/resources/help/en_US/aam/r_tb_variable_prefixes.html)。
* 推出[「演算法模型」的「特徵排除」](https://marketing.adobe.com/resources/help/en_US/aam/trait-exclusion-algo-models.html)，對使用[「角色存取控制」](https://marketing.adobe.com/resources/help/en_US/aam/c_administration.html)的客戶造成問題。建立新模型時，如果只選取可存取的資料來源，可在 **「排除」**視窗查看相應的特徵，但如果同時選取其他無法存取的資料來源，就會看到空白清單。請取消選取無法存取的資料來源，以查看特徵。(AAM-42380)
